# Project Summary

## 市場

フリーランス、個人事業主、小規模事業者向けの請求・見積業務市場を対象にする。既存の会計SaaSは高機能だが、初期設定や運用が重く、スマホ中心で素早く見積から請求まで進めたい層には過剰になりやすい。

## 競合

- freee
- マネーフォワード クラウド
- Misoca
- Stripe Invoicing
- Google スプレッドシート / Excel テンプレート

## MVP

- 顧客管理
- 見積作成
- 見積履歴
- 請求書作成
- 請求書履歴
- PDF出力
- Supabase保存
- スマホファーストUI

## 今後の機能

- 認証
- Stripe課金
- 顧客別詳細ページ
- 見積から請求書への履歴連携強化
- 入金管理
- PDFテンプレート選択
- 月次売上レポート
- チーム・権限管理
- 本番向けRLS設計

## 収益モデル

- 月額980円のサブスクリプション
- 無料プランは作成件数や保存件数を制限
- 有料プランでPDF出力、履歴保存、Supabase同期、請求書管理を開放
- 将来的にチームプランやテンプレート追加課金を検討
